Установка на Ubuntu:<br>
На официальном сайте документации «Docker» есть страница с описанием различных способов установки.<br>
Установим «Docker» с помощью скрипта:<br>
```bash
curl -fsSL https://get.docker.com -o get-docker.sh***
sudo sh get-docker.sh
```
Добавим текущего пользователя в группу «docker»:<br>
```bash
usermod -aG docker $USER
```
```-a``` – добавить в группы, указанные в параметре «-G» не удаляя из групп, в которых пользователь уже состоит.<br>
Сокет-демон «Docker» (```/var/run/docker.sock```) по умолчанию принадлежит пользователю «root» и группе «docker». Если пользователь входит в группу «docker», он получает право на запись в этот сокет. Любой, кто может писать в этот сокет, может отдавать команды демону «Docker».<br>
Лучше не выполнять на проде, так как пользователь фактически получает права «root» системы – можно запустить контейнер с монтированием всей файловой системы (ФС) хоста и делать с ней что угодно.
