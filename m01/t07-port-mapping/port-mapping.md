```
docker run -d -p 8080:80 nginx
```
Слева порт хоста, а справа порт контейнера.

Если в браузере VM ввести ```http://localhost:8080```, то откроется приветственная страница «nginx».
Если же контейнер находится на VM (с сетевым мостом), а браузер - на хосте, на котором запущена VM, то приветственную страницу «nginx» можно найти по адресу:<br>
```ip-адрес контейнера:порт хоста, связанный с контейнером```
