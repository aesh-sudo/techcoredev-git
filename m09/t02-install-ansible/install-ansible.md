```pipx``` – устанавливает Python-приложения в изолированные окружения, не затрагивая системные пакеты.
```bash
sudo apt install -y pipx
pipx ensurepath # Добавляет pipx в PATH (может потребоваться перезапуск терминала)
```
Далее устанавливаем сам «Ansible»:
```bash
pipx install ansible-core
```
Приложение должно установиться по адресу:
```
~/.local/bin
```

Чтобы система видела установленные через ```pipx``` приложения, необходимо добавить адрес:
```
~/.local/bin
```
в переменную ```PATH```.

Проверка установленной версии «Ansible»:
```bash
ansible --version
```
