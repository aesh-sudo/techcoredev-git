Чтобы добраться до пода извне кластера, «k8s» предлагает три основных механизма. Они идут по пути усложнения настройки, но увеличения удобства и экономии ресурсов:

```NodePort``` (Уровень ```L4```):<br>
Открывает один и тот же статический порт (в диапазоне ```30000–32767```) на каждой Ноде (VM) кластера.<br>
Маршрут трафика:
```
Клиент – IP_любой_Ноды:3xxxx – Service (ClusterIP) – Pod
```
Плюсы:
* Бесплатно.
* Не требует сложных настроек.

Минусы:
* Необходимо знать IP-адрес конкретной Ноды. Если Нода меняется, ей назначается новый IP-адрес.
* Ограниченное количество портов.
* Проблема безопасности – открытые порты на всех Нодах.
* Нет возможности маршрутизировать трафик по домену или пути. Только по порту.

Используется в основном для разработки и тестирования.

```LoadBalancer``` (Уровень ```L4```):<br>
«k8s» обращается к API облачного провайдера, который создает внешний балансировщик нагрузки.<br>
Маршрут трафика:
```
Клиент – Внешний IP-адрес облачного LB – Service (тип «LoadBalancer») – Pod
```
Плюсы:
* Один стабильный внешний IP-адрес.
* Полностью автоматизировано (создал сервис – получил IP-адрес).
* Скрыта внутренняя структура кластера.

Минусы:
* Цена. Каждый сервис типа ```LoadBalancer``` требует отдельный IP-адрес и балансировщик.
* Работает только в поддерживаемых облачных средах.

```Ingress``` (Уровень ```L7```, ```HTTP/HTTPS```):<br>
Объект «k8s», который представляет из себя набор правил маршрутизации. Читает HTTP-заголовки (host, path) и решает, какому внутреннему Сервису направить запрос. В кластере обязательно должен быть установлен ```Ingress Controller```, который перенаправляет запрос.<br>
Маршрут трафика:
```
Клиент - Внешний IP (часто это «LoadBalancer» для контроллера) - Ingress Controller - Правило Ingress – Service - Pod.
```
Плюсы:
* Один ```Ingress Controller``` и ```LoadBalancer``` для него могут обслуживать множество Сервисов.
* Умная маршрутизация. Может направлять трафик по домену или по пути.
* SSL. Можно настроить HTTPS-сертификаты в одном месте.

Минусы:
* Требует первоначальной настройки и установки ```Ingress Controller```.
* Работает только с ```HTTP/HTTPS```.
